This directory contains all of the data and scripts reqired for 
reproducing the analysis and figures presented in the manuscript, 
"Differentially tolerant and metabolically distinct subpopulations 
in shaken batch cultures of Pseudomonas aeruginosa". The manuscript 
has been submitted to NPJ Biofilms in January 2023.

1.  Create an anaconda enviroment from the ./Scripts/compare_plank_agg_GFP.yaml
